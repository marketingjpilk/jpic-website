Drop news images here as .webp files.
Name format: [article-id]-cover.webp, [article-id]-img1.webp, [article-id]-img2.webp, etc.

Examples:
  001-cover.webp
  001-img1.webp
  001-img2.webp
  002-cover.webp

To add a new article:
1. Add a new object to news/news.json with the next id ("002", "003", etc.)
2. Drop images here named with the article id prefix
3. Git commit and push — Cloudflare Pages deploys automatically
